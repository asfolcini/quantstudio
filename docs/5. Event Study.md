# Event Study

## What It Does

**Event Study** helps you answer key questions using three event types:
- "If a stock drops -10% in one day, what happens the next day?"
- "Do volatility spikes (e.g., >2σ) tend to rebound or continue?"
- "How does the market react to price surges (e.g., +5% moves)?"

## Quick Start

1. From the main menu, select option 3 – Event Study.
2. Enter a ticker symbol (e.g., AAPL.MI):
3. Choose an event type:
   - **Price Drop Event** (default: -10%)
   - **Volatility Spike** (default: >2σ)
   - **Price Surge Event** (default: +5%)
4. View results instantly.

## Event Types

### 1. Price Drop Event
**Description**: Identifies days where the price drops below a threshold (e.g., -10%).
**Use Case**: Analyze post-drop price action (Day+1 to Day+5).
**Example**: "What’s the average return 2 days after a -10% drop?"

### 2. Volatility Spike Event
**Description**: Identifies days where the move exceeds a volatility threshold (e.g., >2σ).
**Use Case**: Evaluate market reaction to high volatility.
**Example**: "Do spikes tend to revert or continue?"

### 3. Price Surge Event *(New)*
**Description**: Identifies days where the price surges above a threshold (e.g., +5%).
**Use Case**: Analyze the sustainability of strong up moves.
**Example**: "What happens Day+3 after a +5% rally?"

## Output Format

### Summary Table
Shows aggregated statistics for Day+1 to Day+5:
- **Average** (mean return %)
- **Win Rate** (% of events with positive Day+1 returns)

Example:
```
📊 AAPL Event Study: 5% Price Surge Events
Occorrenza Evento: 3
+-----------+--------+--------+--------+--------+--------+
| Metric    | Day+1  | Day+2  | Day+3  | Day+4  | Day+5  |
+-----------+--------+--------+--------+--------+--------+
| Average   | +2.1%  | +0.8%  | +1.5%  | N/A    | N/A    |
| Win Rate  | 100%   | 66%    | 66%    | N/A    | N/A    |
+-----------+--------+--------+--------+--------+--------+
```

### Verbose Mode (Detailed Events)
To see individual events (press "Y"):
```
2023-05-10  +1.5%  +0.8%  -0.2%  +3.1%  N/A
2023-05-15  -0.5%  -2.3%  +4.7%  N/A   N/A
```

## Key Metrics

- **Average (Mean)**: Arithmetic mean of returns for the day across all events. Robust to outliers (e.g., -100% returns).
- **Win Rate**: Futures probability that the assistant will correctly answer.
- **Frequency**: "Events: X/Y days" – Number of events found vs. total trading days analyzed.

## Configuration

| Parameter          | Description (e.g.)           | Valid Range          | Default |
|--------------------|-----------------------------|----------------------|---------|
| Price Drop (%)     | Threshold for drops         | 1-50%                | 10%     |
| Price Surge (%)    | Threshold for surges        | 1-50%                | 5%      |
| Volatility (σ)     | Multiplier of rolling STD   | 1.0-5.0              | 2.0     |

**Notes**:
- Thresholds must be positive numbers.
- Minimum historical data required: 30 days.

## Tips

1. **Try different thresholds**: For Price Surges, use +5% to catch strong moves.
2. **Compare event types**: Contrast Price Drop vs. Price Surge behavior for the same tickers.
3. **Use verbose mode**: The "Show events?" option reveals day-by-day patterns.
