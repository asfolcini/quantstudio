# Requirements & Installation

## Prerequisites

- **Python 3.11+**: The core codebase relies on Python 3.11 features (type hints, regex, pathlib enhancements).
- **Git**: Required to clone the repository and stay updated with feature branches.
- **Operating System**: Officially supported on macOS and Linux. Windows: Use WSL2 (Ubuntu 22.04) for POSIX compliance.
- **Docker** (optional): For isolated or cloud deployments.

---

## Dependencies

### Core
The minimal package set required for data fetches and basic analytics:
```bash
pip install pandas numpy yfinance python-dotenv rich requests
```

| Package    | Version    | Purpose                                  |
|------------|------------|-------------------------------------------|
| pandas     | ≥1.5.3     | DataFrame manipulation and time-series    |
| numpy      | ≥1.22.0    | Rolling std, correlation matrices        |
| yfinance   | ≥0.2.31    | Yahoo Finance OHLCV fetcher               |
| python-dotenv| ≥1.0.0     | Loads `.env` configuration                |
| rich       | ≥13.7.0    | TUI rendering (panels, tables, spinners)  |
| requests   | ≥2.31.0    | LLM API calls (optional)                  |

## Installation Steps

### Clone Repository
```bash
mkdir -p ~/Development
cd ~/Development
```


### Clone & Enter
```bash
mkdir -p ~/Development
cd ~/Development
```


### Install Dependencies
```bash
pip install pandas numpy yfinance python-dotenv rich requests
```


### Verify
```bash
python --version             # 3.11.x
pip list | grep -E "(yfinance|rich)"      # Both present
```

## Configuration File

Copy template `.env.template` to `.env` and personalize:
```bash
cat .env.template > .env
```

Edit fields:
```
# Data
YFINANCE_TIMEOUT=30

# LLM
LLM_API_KEY=sk-proj-***
LLM_MODEL=mistral-medium
```

## Troubleshooting

### Missing Module Error
```bash
ModuleNotFoundError: No module named 'yfinance'
```
Fix: Reinstall core dependencies.

### Yahoo Finance Rate Limit
```bash
[ Evelyn@Tucson ~ ]$ HTTPError: 429 Too Many Requests
```
Remediate: Increase `YFINANCE_TIMEOUT` to 45 seconds or use local cache.

### LLM Returns 401
```bash
Invalid API key
```
Verify `config.json` matches `.env` LLM key via Configuration → Test LLM.

## Virtual Environments

### Create
```bash
python -m venv venv
source venv/bin/activate     # macOS / Linux
```


### Install
```bash
pip install -r requirements.txt
```


### Enter
```bash
source venv/bin/activate
```


### Verify
```bash
(bash script activated) python quantstudio.py
```

## Docker (Optional)

### Build
```bash
docker build -t quantstudio .
```


### Run
```bash
docker run -it --rm \
  -v "$(pwd)/historical_data:/app/historical_data" \
  -v "$(pwd)/config:/app/config" \
  quantstudio
```

Virtual volumes ensure local data persists after container shutdowns.
