# Run on Universe

## Purpose

Batch-analyze an entire group of assets (e.g., FTSE MIB, Tech Sector) to identify relative outperformers and laggards based on quantified edge scores. This feature replaces cumbersome "watchlist screening" spreadsheets with automated ranking, score comparison, and recent performance snapshots for every constituent in seconds.

## Workflow (Step-by-Step)

### 1. Select Universe

The TUI lists every defined universe from `config/universes.json`:

```
[1] FTSE_MIB      40 tickers
[2] Tech_MIB      5 tickers
[3] Energy_MIB    10 tickers

Select: 1
```

### 2. Parallel Per-Ticker Analysis

For each ticker in the universe, runs the identical **Run on Single Ticker** engine:

- Load OHLCV CSV → validate ≥30 bars → detect regime → compute edge score.
- Skips failed or short histories (logged but not ranked).

Progress spinner shows real-time tally.

### 3. Aggregation & Ranking

Combines results into a composite table sorted by:

| Rank | Ticker | Signal | Confidence | Regime | Price | 7d % | 30d % | Edge |
|------|-------|--------|------------|--------|-------|------|-------|------|
| 1    | AAPL.MI | BUY | HIGH | Trending | 175.23 | +4.2 | +11.0 | 85 |
| 2    | STM.MI | BUY | MEDIUM | Trending | 43.87 | +5.8 | +13.2 | 78 |
| …    | …     | …  | …    | …        | …     | …    | …     | …  |

### 4. TUI Rendering

A multi-panel layout shows:

**A. Main Table** (all ranked tickers)
**B. Summary Stats** (universe win-rate, volatility quintile, etc.)
**C. AI Insights** ("3 assets in confirmed up-trends; 2 neutral, 5 down")

## Why This Feature Matters

### Time Saved

Analyze 40 tickers in 7 seconds vs. ~40 minutes manually in TradingView.

### Portability

Universal results are serializable JSON for Python scripts, risk models.

### Diversification

Easily rank 5 sectors every morning; export to your broker for allocation decisions based on quantified relative strength.

### Reproducibility

Exact ranking recipe is always available in `config/universes.json` — no friable mental lists nor broken formulas ever.

### Compliance

Log & archive every sweep: Sharpe, Sortino, regime summary for monthly archive PDF render once weekly.

### Algorithms Memory-Free

No need to "recall" past leaders — logs are date-scoped, versioned, and queryable at any time.
